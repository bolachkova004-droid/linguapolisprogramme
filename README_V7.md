# Linguapolis v7 — наставник Irene

В этой версии добавлен персональный наставник Irene:

- приветственная карточка;
- обратная связь после каждого ответа;
- отдельные рекомендации: исправить, развернуть, усилить лексику или похвала;
- подсказки для ответа из 2–3 предложений;
- итог Irene после завершения урока;
- рекомендация Irene в аналитике;
- фотография наставника оптимизирована для сайта.

## Установка

Загрузите все файлы и папки из этого архива в корень GitHub-репозитория с заменой старых файлов.

`config.js` намеренно не включён — существующие ключи Supabase сохранятся.
