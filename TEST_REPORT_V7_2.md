# Linguapolis v7.2 — исправление запуска

Причина: приложение подключало обработчики кнопок только после загрузки `data.json`. Если файл не загрузился, JavaScript останавливался до привязки кнопок, поэтому вкладки входа, регистрация и демо-режим не реагировали.

Исправлено:
- обработчики кнопок подключаются до загрузки внешних данных;
- данные уроков встроены в `index.html` как резервная копия;
- при отсутствии или ошибке `data.json` приложение продолжает запускаться;
- обновлена версия кэша Service Worker;
- обновлены версии CSS и JavaScript для принудительного обновления браузером;
- проверен синтаксис JavaScript и структура JSON.
