# Как установить v7

1. Распакуйте архив.
2. В GitHub откройте репозиторий `linguapolisprogramme`.
3. Нажмите **Add file → Upload files**.
4. Загрузите все файлы и папки из архива.
5. Разрешите заменить старые файлы.
6. Нажмите **Commit changes**.
7. Подождите 1–3 минуты.
8. Откройте сайт и обновите страницу без кэша.

Не удаляйте существующий `config.js`.
