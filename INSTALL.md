# Как установить v5 без проверки на компьютере

1. Скачайте архив `linguapolisprogramme-v5-update.zip` и распакуйте его.
2. Откройте репозиторий `linguapolisprogramme` на GitHub.
3. Нажмите **Add file → Upload files**.
4. Перетащите все файлы и папку `assets` из архива.
5. Когда GitHub спросит о замене файлов, подтвердите.
6. **Не удаляйте и не заменяйте ваш заполненный `config.js`.**
7. Нажмите **Commit changes**.
8. Подождите 1–3 минуты и откройте сайт.
9. Обновите страницу без кэша: `Command + Shift + R` на Mac или `Ctrl + Shift + R` на Windows.

Новые картинки не появятся, если папка `assets` не была загружена целиком.
