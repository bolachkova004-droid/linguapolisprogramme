# Linguapolis Sim — improved MVP

Обновлённая версия статического симулятора разговорного английского.

## Что добавлено

- журналирование действий в IndexedDB;
- база ответов с текстом, метриками, обратной связью и изменением навыков;
- оценки уместности, лексики, структуры и беглости;
- уровни навыков, пороги 25/50/75/100 и разблокировки;
- аналитика по ответам, средним оценкам, урокам и серии занятий;
- локальная админка: метрики, журнал, редактирование навыков, экспорт и сброс;
- полноценный сценарий завершения урока с защитой от повторной награды;
- кнопки «Домой» и «Выход»;
- более спокойные минималистичные персонажи без видео и мультяшной стилистики;
- адаптивный интерфейс для телефона и компьютера.

## Запуск

Из-за загрузки `data.json` браузеру нужен локальный сервер:

```bash
cd linguapolis-sim-improved
python -m http.server 8000
```

Откройте `http://localhost:8000`.

## Хранение данных

Демо работает без бэкенда:

- профили и текущий урок — `localStorage`;
- ответы и action log — `IndexedDB`;
- при отсутствии IndexedDB используется локальный fallback.

Для реального сервиса перенесите записи на сервер. Стартовая PostgreSQL-схема лежит в `backend/schema.sql`. Локальная админка не является безопасной production-админкой: права доступа и расчёт наград должны проверяться сервером.
