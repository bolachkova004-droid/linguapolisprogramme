# Linguapolis Life v5

Игровой симулятор разговорного английского для публикации на GitHub Pages.

## Что появилось в v5

- полностью обновлённый светлый life-sim дизайн;
- 10 оригинальных аватаров для аккаунтов;
- 6 иллюстрированных наставников;
- выбор аватара во время регистрации и смена в профиле;
- XP, уровни, монеты и серия занятий;
- три ежедневных задания с отдельной наградой;
- восемь достижений;
- карта из шести разговорных локаций;
- постепенное открытие новых уроков;
- настроение персонажа;
- анимации наград и повышения уровня;
- сохранена совместимость с прогрессом из v4.

## Самое безопасное обновление существующего сайта

Если вы уже заполнили `config.js` данными Supabase, **не заменяйте этот файл**.

Загрузите в GitHub содержимое архива обновления:

- `index.html`
- `style.css`
- `script.js`
- `auth.js`
- `data.json`
- папку `assets`
- папку `backend` — необязательно, для базы Supabase

Старый `config.js` оставьте в репозитории без изменений.

## Настоящая регистрация

Регистрация продолжит работать с уже подключённым Supabase. Новый аватар сохраняется в `user_metadata.avatar_id`.

Если схема Supabase v4 уже была установлена, можно повторно запустить `backend/schema.sql`: команда `add column if not exists` безопасно добавит поле `avatar_id`.

## Где хранятся данные сейчас

- Supabase: аккаунт, email, имя, код класса, аватар и роль.
- Браузер: игровой прогресс, ежедневные задания, достижения, ответы и события.
- `backend/schema.sql`: структура для будущей полной облачной синхронизации.

## Важно

В GitHub разрешено публиковать только Supabase publishable/anon key. Никогда не добавляйте `service_role` или secret key.
