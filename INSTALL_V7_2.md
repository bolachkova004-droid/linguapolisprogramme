# Установка v7.2

1. Загрузите в корень репозитория все файлы и папки из архива.
2. Не удаляйте существующий `config.js` с настройками Supabase.
3. Подтвердите замену файлов и нажмите Commit changes.
4. Подождите публикацию GitHub Pages.
5. Откройте сайт в приватном окне. Если старая версия осталась, очистите данные сайта или выполните жёсткое обновление.
